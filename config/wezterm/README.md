# WezTerm 設定

## SSH 接続先の設定方法

このリポジトリーの `wezterm.lua` は `wezterm.default_ssh_domains()` を使って `~/.ssh/config` から SSH 接続先を自動的に読み込みます.
接続先のホスト名・アドレス・ユーザー名は `~/.ssh/config` に記述し, このリポジトリーには含めません.

### ~/.ssh/config の記述例

```
Host myserver
  HostName 192.168.1.100
  User myuser
```

## SSH マルチプレクサーへの接続

`~/.ssh/config` に Host を定義すると, 以下のコマンドで接続できます.

```
wezterm connect SSHMUX:myserver
```

`SSHMUX:<host>` はサーバー上で動いている WezTerm マルチプレクサーに接続します.
サーバー上にワークスペースが 2 つ以上ある場合は, 接続直後に選択画面が自動表示されます.

## ショートカットキー一覧

### タブ

| キー | 動作 |
|------|------|
| Shift + ← | 左のタブへ移動する |
| Shift + → | 右のタブへ移動する |
| Shift + Space | シェルを選択して新規タブを開く |
| Shift + Delete | 現在のタブを閉じる |

### ペイン (画面分割)

ペインはウィンドウ内を縦横に分割した各領域です. 1 つのタブ内で複数のシェルを並べて表示できます.

| キー | 動作 |
|------|------|
| Ctrl + Shift + → | 右にペインを分割する (シェル選択あり) |
| Ctrl + Shift + ← | 左にペインを分割する (シェル選択あり) |
| Ctrl + Shift + ↑ | 上にペインを分割する (シェル選択あり) |
| Ctrl + Shift + ↓ | 下にペインを分割する (シェル選択あり) |
| Ctrl + Q | 現在のペインを閉じる |
| Ctrl + F | 現在のペインをタブ全体に拡大/縮小する |
| Ctrl + J | ペイン選択画面を表示する (`uijknm` で対象を選択, Esc でキャンセル) |
| Ctrl + Alt + ← | ペインの幅を左方向に 1 段階広げる |
| Ctrl + Alt + → | ペインの幅を右方向に 1 段階広げる |
| Ctrl + Alt + ↑ | ペインの高さを上方向に 1 段階広げる |
| Ctrl + Alt + ↓ | ペインの高さを下方向に 1 段階広げる |

### ワークスペース (独立したセッション)

ワークスペースはタブ群をまとめた独立したセッションです. プロジェクトごとに切り替えて使えます.

| キー | 動作 |
|------|------|
| Ctrl + Shift + L | ワークスペース一覧を表示して切り替える |
| Ctrl + Shift + W | 新しいワークスペースを名前をつけて作成する |

### その他

| キー | 動作 |
|------|------|
| Ctrl + S | 無効 (誤操作による端末フリーズを防ぐため)
