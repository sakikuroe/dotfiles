# 散文構造のルール

返答, Issue, PR, ドキュメントなど, あらゆる文章は整理された散文として書くこと. 箇条書きを主体にせず, 背景・課題・解決策のロジックが段落を通じて自然に流れるよう論理を展開する. 表・図・数式など理解を助けるものは積極的に活用してよい.

以下の例が示すスタイルに従うこと.

```markdown
フィボナッチ数列は $F_0 = 0,\ F_1 = 1$ を初期値とし, 漸化式 $F_n = F_{n-1} + F_{n-2}$ によって定義される. この漸化式は隣接する 2 項の関係であるため, 状態ベクトル $\mathbf{v}_n = (F_{n+1},\ F_n)^\top$ を導入すると, 漸化式を行列の積として書き直すことができる.

$$\mathbf{v}_n = A\mathbf{v}_{n-1}, \quad A = \begin{pmatrix} 1 & 1 \\ 1 & 0 \end{pmatrix}$$

この形式に帰着できると, $n$ ステップ後の状態が $\mathbf{v}_n = A^n \mathbf{v}_0$ と表せるため, 問題は $A^n$ の計算に還元される. $A^n$ を効率よく求めるために行列を対角化する. $A$ の固有値は特性方程式 $\lambda^2 - \lambda - 1 = 0$ の解であり, $\varphi = (1 + \sqrt{5})/2$ および $\psi = (1 - \sqrt{5})/2$ の 2 値が得られる. これらを用いて $A = PDP^{-1}$ と分解すると $A^n = PD^nP^{-1}$ が成り立ち, 成分を整理することで一般項の閉じた式が導かれる.

$$F_n = \frac{\varphi^n - \psi^n}{\sqrt{5}}$$

$|\psi| < 1$ であるから $\psi^n$ は $n \to \infty$ でゼロに収束し, $F_n$ は $\varphi^n / \sqrt{5}$ に漸近する. 線形代数の道具を経由することで, 再帰的な定義から出発しながら指数関数で直接表現できる閉形式が得られるという点が, この導出の本質である.
```
