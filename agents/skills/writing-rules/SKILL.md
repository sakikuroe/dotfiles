description: 日本語の文体・記述ルールと Markdown の書き方を定めたスキル. 日本語で返答, ドキュメント, コメント, コミットメッセージを書くとき, 英語から日本語へ翻訳するとき, または Markdown を記述するときは必ず参照する.
---

日本語で文章を書く場面では `japanese_writing_rule.md` と `japanese_tone.md` を参照します. 返答の構造・散文スタイルを確認する場合は `response_format.md` を, Markdown を使う場合は `markdown_rule.md` を参照します. 英語から日本語へ翻訳する場合は上記に加えて `english_to_japanese.md` も参照します.

- [response_format.md](./references/response_format.md): 返答スタイルのルール. 箇条書きを主体にせず, 散文として論理を展開する.
- [japanese_writing_rule.md](./references/japanese_writing_rule.md): 句読点, スペース, 表現, 漢字の使い分けなど, 日本語の構造的な記述ルール.
- [japanese_tone.md](./references/japanese_tone.md): 文体と語彙の方針. トーン, 丁寧さ, 使用する語彙の基準.
- [english_to_japanese.md](./references/english_to_japanese.md): 英語から日本語へ翻訳する際のルール.
- [markdown_rule.md](./references/markdown_rule.md): Markdown の使用可能な記法と書き方のルール.
