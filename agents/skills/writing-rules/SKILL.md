この skill は, このリポジトリーで日本語または Markdown を書くターン全体に適用します.
返答, ドキュメント, コメント, コミットメッセージなど, 文章を記述するあらゆる場面において, 本 skill の参照規則を確認して進めてください.

参照規則は次の通りです.

- [japanese_writing_rule.md](./references/japanese_writing_rule.md): 日本語を書く際の構造的な記述ルールを示します. 句読点, スペース, 表現, 漢字の使い分けなど, 記述の形式を定めます.
- [japanese_tone.md](./references/japanese_tone.md): 日本語を書く際の文体と語彙の方針を示します. 文のトーン, 丁寧さ, 使用する語彙の基準を定めます.
- [english_to_japanese.md](./references/english_to_japanese.md): 英語から日本語へ翻訳する際のルールを示します.
- [markdown_rule.md](./references/markdown_rule.md): Markdown を書く際のルールを示します.
